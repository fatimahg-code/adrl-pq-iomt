# ADRL-PQ Authentication for IoMT

This repository contains the implementation of an adaptive authentication framework
for IoMT systems using Deep Reinforcement Learning (DRL).

## Features
- Adaptive TLS 1.3 authentication
- Post-quantum cryptography (Dilithium, SPHINCS+)
- Attack simulation (MITM, Replay, Forgery, Tampering, Downgrade)
- DRL-based policy (PPO)

## Requirements
- Python 3.12
- Stable-Baselines3
- Gym
- PyTorch

## Run
pip install -r requirements.txt
python smartphone_medical_emulator.py
